# tic_tac_toe

Practice project with React and Redux and Jest.

## Setup

### Prerequisites

* [Node](https://nodejs.org/en/)

### Run

* From root run `npm start`.
* In browser navigate to `localhost:8080` to view the site and play the game.

## Testing

* `npm run test`.
